# 🛒 Amazon Search Results Scraper (Playwright)

## 🚀 Project Overview

This project scrapes product listings from Amazon search results pages using Playwright.

The scraper automatically:
- Searches for a user-defined keyword
- Extracts product data from each page
- Navigates through all available pages
- Saves structured results to a CSV file

This project demonstrates dynamic website scraping using browser automation.

---

## 🌐 Target Website

Amazon (Search Results Pages)

Example:
https://www.amazon.com/s?k=laptop

---

## 🛠 Technologies Used

- Python 3
- Playwright
- Pandas
- Chromium Browser Automation

---

## ✨ Features

- Dynamic website scraping
- Automatic pagination handling
- Extracts:
  - Product Title
  - Price
  - Rating
  - Review Count
  - Product URL
- User input for search keyword
- Structured CSV export
- Random delay to simulate human behavior
