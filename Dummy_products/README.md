# 🛒 E-commerce Product API Scraper (Python)

## 🚀 Project Overview

This project extracts product data from the DummyJSON public API.

The scraper automatically handles pagination using limit and skip parameters, retrieves all available products, and exports structured data into a CSV file.

This project demonstrates API-based data extraction using Python.

---

## 🌐 Data Source

DummyJSON Public API  
https://dummyjson.com/products

---

## 🛠 Technologies Used

- Python 3
- requests
- CSV module

---

## ✨ Features

- API-based data extraction
- Automatic pagination handling
- Structured product data collection
- Clean CSV export
- Modular code structure
- User-Agent header handling

---

## 📊 Data Extracted

The scraper collects:

- Product ID
- Title
- Brand
- Category
- Price
- Discount Percentage
- Rating
- Stock
- Description
- Thumbnail URL
