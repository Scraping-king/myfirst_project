# Amazon Full Product Scraper (Selenium)

## 📌 Project Description
This project searches Amazon by keyword and scrapes:
- Product Title
- Price
- Rating
- Product Link
- Timestamp

It navigates through multiple pages automatically and saves data into a CSV file.

## 🚀 How To Run

1. Install dependencies:
   pip install -r requirements.txt

2. Make sure ChromeDriver is installed and matches your Chrome version.

3. Run:
   python main.py

4. Enter a keyword like:
   laptop
   headphones
   gaming mouse

## ⚠️ Note
Amazon may block scraping attempts.
Use responsibly for educational purposes only.
