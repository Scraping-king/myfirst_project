# 📚 Quotes Web Scraper (Python + BeautifulSoup)

## 🚀 Project Overview

This project scrapes quotes, authors, and tags from the website:

http://quotes.toscrape.com

The scraper automatically navigates through all available pages, extracts structured data, and saves the results into a CSV file.

---

## 🛠 Technologies Used

- Python 3
- requests
- BeautifulSoup4
- CSV module

---

## ✨ Features

- Handles multi-page pagination
- Extracts:
  - Quote text
  - Author name
  - Tags
- Automatic page detection (stops when no more pages exist)
- Clean CSV export
- Modular project structure
